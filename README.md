# Chinook Cyber Homepage

A modern, minimalist splash page acting as a portal to the Chinook Cyber ecosystem. 
It features a split-screen design linking to our two core applications.

## Ecosystem

- **[Cyber Planner](https://planner.chinookcyber.com/)**: Interactive OAKOC-based cyber planning and maneuver workspace.
- **[Threat Group Browser](https://browser.chinookcyber.com/)**: Curated ATT&CK intrusion set corpus and threat landscape.

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- [Tailwind CSS 4](https://tailwindcss.com/)
- Lucide React Icons

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Design System

The application utilizes a dark-mode first design system with custom glassmorphism utilities to maintain visual consistency with the overarching Chinook Cyber applications. Color variables (e.g., `--color-key-terrain`, `--color-avenue`) are directly mapped from the Cyber Planner's standard OAKOC palette.
